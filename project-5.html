Project 5 - Jupyter Notebook

Many platforms can intelligently understand the flowing data generated by accounts, scores, or other miscellaneous items. Thus, here I am attempting to show how I can extract data, manipulate the data, and present the outcomes in an organized way for a complete understanding.

Project Overview

Within this file, I am extracting built-in data inside the Jupyter Notebook Sklearn dataset and showing how it is possible to calculate the data for visual inspection accurately. Thus, the differences between the numbers are easily seen when that data is split up and divided into different categories. Linear regression, predictions, accuracy reports, and polynomial regression are used to show these differences.
