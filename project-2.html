Project 2 - Chatter App

I have created several apps using Xcode, all written in Swift. Although I could run and deploy these applications successfully, all apps are only in testing modes. Meaning, you won’t find these applications in the App Store. Also, to gain experience using Xcode’s platform, Udemy online courses played a crucial part in obtaining the skills.

Project Overview

The function of this application is to text a friend. Chatter has a Cloud service that can accomplish the following tasks: multiple accounts set up, password set up, and storage of texts.
