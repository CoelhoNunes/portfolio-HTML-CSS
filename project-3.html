Project 3 - Sudoku Generator

I attempted to manipulate and adapt artificial intelligence and software development techniques in several ways. The goal was to branch out to as many areas as possible to develop meaningful software. I am always working on mini-projects, which means that this area will consistently update with new developments.

Project Overview

Each time the user runs the program, the code will create the 9 x 9 cell in a sudoku cell field and randomly populate the 81 cells with only 18 numbers. Each cell between 1 and 9 will follow the rules of Sudoku, meaning the area will be solvable by the inputted numbers; thus, the user can find a number according to the regulations in each row or column, and the 9 x 9 field can be filled in by the user.
